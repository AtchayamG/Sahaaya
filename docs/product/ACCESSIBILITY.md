# Accessibility Requirements

- WCAG 2.2 AA target for colour contrast, focus, semantics, and keyboard operation.
- Provider, decision, conflict, and verification states include text and icons, not colour alone.
- All controls use visible labels and 44px target size where practical.
- Progress changes and errors use polite live regions.
- Dialog focus is trapped and restored.
- Motion respects `prefers-reduced-motion`.
- Desktop and 390px mobile layouts must pass axe with no serious/critical violations.
