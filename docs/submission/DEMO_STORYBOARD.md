| 1:22-1:45 | Assign and approve | The user accepts or edits every proposed owner. | Explicit human decisions; no external message is sent. |

| 2:08-2:22 | Export | A clear continuity pack is prepared locally. | Safe redactions and no credentials or access codes. |
| 2:22-2:30 | Close | One absence, one aligned circle, no guesswork. | Codex and GPT-5.6 attribution, repository and app URLs in description. |

## Capture rules

- Use 1920x1080 at 100% browser zoom.
- Keep pointer movement deliberate and avoid showing personal browser chrome or secrets.
- Record one uninterrupted golden path after `demo-assert` passes.
- Do not describe Replay output as live model evidence.
